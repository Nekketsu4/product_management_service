Управление складом и продажами для малого бизнеса

Проблема: Малому бизнесу нужен простой и дешевый способ учета товаров, продаж и клиентов.

#### Функционал: python

#### Основные модули
- Каталог товаров (артикул, название, цена, остаток)
- Управление запасами (приход/расход, остатки)
- CRM (клиенты, история покупок)
- Продажи и чеки
- Аналитика (продажи по дням, популярные товары)
- Уведомления о низких остатках

1. Пользователь может добавлять, обновлять, удалять товар в каталог
2. Через каталог мы можем добавлять, отнимать нужное количество определенного товара на складе
3. Склад содержит информацию приходе/ расходе конкретного товара а также его остатке
4. Пользователь может добавлять, удалять, редактировать пользователей а также просматривать историю их покупок
5. На каждого клиента есть детальный обзор продаж а также их чеки
6. Пользователь может выполнить аналитику популярных товаров по дням или в целом самых популярных товаров
